# 디시인사이드 만화 다운로더 스위트 (DC Manga Downloader Suite)

디시인사이드(만갤 등)에서 번역된 만화 게시글을 자동으로 탐색, 파싱 및 다운로드하도록 설계된 고급 멀티스레드 PowerShell 스크립트 모음입니다.

## 🚀 주요 기능

* **강력한 안티봇 우회:** 최신 브라우저 핑거프린팅(Chrome User-Agent, Sec-Fetch 헤더 등)을 사용하여 디시인사이드의 봇 차단(403 Forbidden, 빈 페이지)을 우회합니다.
* **스마트 제목 파싱:** 폴더 이름을 깔끔하게 정리합니다. `번역)` 같은 머리말과 불필요한 부연 설명을 제거하고, 복잡한 화수(예: `2화＆9화`, `1.5화`, 혹은 `29`와 같은 단독 숫자)를 정확히 추출합니다.
* **병렬 다운로드 (제한 해제):** PowerShell Job을 활용해 여러 이미지를 동시에 다운로드하여 속도를 극대화합니다. 스레드 제한을 자유롭게 설정할 수 있습니다.
* **자동 재시도 및 실패 방지 (Fail-Safe):** 네트워크가 끊기거나 일부 이미지 다운로드에 실패하면, 목록에서 삭제하는 대신 `download_list.txt` 파일에 `#RETRY` 태그를 달아두어 나중에 누락된 페이지만 다시 받을 수 있게 합니다.
* **비동기 백그라운드 로깅:** Named Pipe 서버(`Background-Logger.ps1`)를 사용하여 높은 병렬 스레드 작업 중에도 파일 잠금(File-lock) 충돌 없이 안전하게 로그를 기록합니다.
* **DNS 자동 복구:** 호스트 연결에 실패하면 스크립트가 자동으로 DNS를 플러시(`ipconfig /flushdns`)하고 복구를 시도합니다.
* **세션 타이머 및 통계:** 챕터별, 전체 세션별 소요 시간, 성공/실패율, 누적 다운로드 용량을 콘솔에 직관적으로 표시합니다.

## 🛠️ 파일 구성

* `launch.bat` - 프로그램 실행기입니다. `config.yaml`이 없으면 기본 설정값을 자동 생성하고, 백그라운드 로거를 실행한 뒤 메인 메뉴를 띄웁니다.
* `Run-Crawler.ps1` - 설정된 게시판을 스캔하여 '번역'이 포함된 게시글을 찾아 `download_list.txt`에 대기열로 등록합니다.
* `Start-Downloader.ps1` - 다운로드 핵심 엔진입니다. 대기열을 읽어 HTML을 파싱하고, 차단을 우회하며, 이미지를 안전하게 다운로드합니다.
* `Background-Logger.ps1` - 다운로드 속도 저하 없이 로그를 백그라운드에서 처리하는 가벼운 프로세스입니다.
* `config.yaml` - 다운로더의 모든 설정을 제어하는 설정 파일입니다.
* `download_list.txt` - 처리할 대기열 파일입니다 (`[manual_urls]` 및 `[automatic_urls]` 로 나뉨).

## 📖 사용 방법

1. **`launch.bat` 실행** (`.ps1` 파일을 직접 실행하지 마세요).
2. 처음 실행 시 최적화된 기본값이 담긴 `config.yaml` 파일이 자동 생성됩니다.
3. 메뉴에서 원하는 모드를 선택하세요:
   * **1번 (자동 크롤러 & 다운로더):** 설정에 따라 게시판을 스캔하여 목록을 업데이트하고 자동으로 다운로드를 시작합니다.
   * **2번 (수동 다운로더):** 디시인사이드 URL을 직접 붙여넣어 즉시 다운로드합니다.
4. 이미지는 `config.yaml`에 지정된 `DownloadDir` (기본값: `.\Downloads`) 폴더에 저장됩니다.

## ⚙️ 설정 가이드 (`config.yaml`)

`config.yaml` 파일을 수정하여 동작 방식을 세밀하게 제어할 수 있습니다:

### 타겟 설정 (Target Settings)
* **`BoardUrl`**: 탐색할 게시판 주소입니다 (기본: 만갤 추천글 탭).
* **`MaxPages`**: 크롤러가 스캔할 최대 페이지 수입니다.
* **`CrawlOrder`**: `0`으로 설정하면 오래된 글부터 (3페이지 -> 1페이지), `1`로 설정하면 최신 글부터 수집합니다.

### 네트워크 및 성능 (Network & Performance)
* **`MaxConcurrentDownloads`**: 한 챕터 내에서 동시에 다운로드할 이미지 수입니다. (3~5가 안정적이며, 15 이상으로 설정 시 디시인사이드 IP 차단이 발생할 수 있습니다).
* **`RateLimitSeconds`**: 챕터와 챕터 사이의 대기 시간입니다. 서버 과부하 및 IP 밴을 방지합니다.
* **`DNSAutoRepair`**: `True` 설정 시 네트워크 연결 오류 발생 시 자동으로 DNS를 플러시합니다.

### 파일 설정 (File Settings)
* **`DownloadDir`**: 이미지가 저장될 기본 경로입니다.
* **`RenameFilesSequential`**: `True` 설정 시 파일 이름을 `001.jpg`, `002.jpg` 등으로 일괄 변경하여 완벽한 정렬을 보장합니다.
* **`LogLevel`**: `"Verbose"`는 모든 이미지 성공 여부를 기록하며, `"Error"`는 실패 로그 및 요약만 기록합니다.

## ⚠️ 문제 해결: 재시도(Retry) 시스템
챕터 다운로드가 완료되었으나 일부 이미지가 실패했거나, 403 오류가 발생한 경우 콘솔에 **`[FLAGGED]`** 메시지가 출력됩니다.
이 경우 스크립트는 해당 링크를 지우지 않고 `download_list.txt` 내에서 `#RETRY https://...` 형태로 변경하여 보존합니다.
**이 링크를 삭제하지 마세요.** 나중에 다운로더를 다시 실행하면 스크립트가 알아서 해당 게시글의 누락된 이미지만 다시 다운로드합니다.
