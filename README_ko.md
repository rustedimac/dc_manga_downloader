# DC 만화 다운로더

디시인사이드(DCInside)의 만화 번역 게시글을 수집하고 카탈로그화하기 위해 설계된 엔터프라이즈급 고안정성 다운로더 시스템입니다. 

파워 유저를 위해 강력하게 설계되었지만, 누구나 쉽게 사용할 수 있습니다. **코딩이나 커맨드라인 지식이 전혀 필요 없습니다.** `launcher.bat`을 더블 클릭하여 대화형 메뉴에 접속하기만 하면 됩니다. 이 스위트는 자동으로 제목을 지능적으로 분석하고, 기록을 추적하며, 시리즈의 별칭을 관리하고, 누락된 화수를 탐색하며, 전체 컬렉션의 깔끔한 로컬 데이터베이스를 유지 관리합니다.

## ✨ 주요 기능

* **🎮 인터랙티브 런처:** 100% 메뉴 기반으로 작동합니다. PowerShell 코드를 건드리거나 터미널을 수동으로 열 필요가 없습니다.
* **⏰ 백그라운드 스케줄러 (자동 크롤링 및 다운로드):** 사용자가 다른 작업을 하거나 자는 동안에도 백그라운드에서 조용히 게시판을 모니터링하여, 선호하는 시리즈의 새 챕터가 올라오면 즉시 자동으로 다운로드합니다.
* **🚀 초고속 멀티스레딩:** 게시물당 최대 15개의 이미지를 동시에 다운로드하여 대역폭을 극대화하고 대기 시간을 최소화합니다.
* **🧠 초정밀 제목 파싱:** 지저분한 게시물 제목에서 '시리즈 이름'과 '챕터(화수)'를 자동으로 추출하며, 불필요한 텍스트, 추천 유도 멘트, 꼬리말 괄호 등을 완벽하게 걸러냅니다.
* **🔗 데이지 체인 크롤링:** 게시물 내의 "이전 화" 또는 "다음 화" 링크를 감지하고 자동으로 추적하여 카탈로그의 누락된 화수를 채워 넣습니다.
* **📂 스마트 별칭(Alias) 라우팅:** 지저분하거나 일관성 없는 시리즈 이름을 단일하고 깔끔한 "대표 이름(Operator Name)"으로 매핑합니다. 해당 시리즈의 모든 다운로드는 자동으로 지정된 폴더로 라우팅됩니다.
* **🛡️ 중복 방지 및 데이터 보호:** * 정확히 동일한 URL을 두 번 다운로드하는 것을 방지합니다.
  * **원자적 파일 쓰기(Atomic File Writes)** 및 **프로세스 간 Mutex 잠금**을 지원합니다. 백그라운드 스케줄러가 실행 중인 상태에서 카탈로그를 조작하거나 실수로 엑셀을 켜 두더라도 데이터베이스가 절대 손상되지 않습니다.
* **📁 원클릭 라이브러리 접근:** 런처 메뉴에서 정리된 다운로드 디렉토리(윈도우 파일 탐색기)를 즉시 엽니다.

## 🚀 빠른 시작 (Quick Start)

1. **실행:** `launcher.bat`을 더블 클릭합니다. (최초 실행 시 `config.yaml` 설정 파일과 필요한 폴더 구조가 자동으로 생성됩니다).
2. **설정 (선택 사항):** 텍스트 편집기로 `config.yaml`을 열어 다운로드 저장 경로, 스레드 수치 또는 속도 제한을 맞춤 설정할 수 있습니다.
3. **자동화 또는 수동 다운로드:** 백그라운드 스케줄러를 켜서 시스템이 알아서 다운로드하게 두거나, 인터랙티브 스캐너 메뉴를 사용하여 특정 만화를 수동으로 검색하세요.

## ⚙️ 시스템 구조
모든 핵심 로직은 `\core\` 폴더에 포함되어 있으며, `launcher.bat`에 의해 완벽하게 제어됩니다. 아래 스크립트들을 직접 실행할 필요는 없습니다:
* `Start-Downloader.ps1`: 멀티스레드 다운로드 엔진.
* `Search-Scanner.ps1`: 데이터베이스 관리자 및 게시판 크롤러.
* `Clean-Catalog.ps1`: 데이터베이스 상태 점검 및 중복 제거 유틸리티.
* `Background-Logger.ps1`: 백그라운드 이벤트 로깅 시스템.
