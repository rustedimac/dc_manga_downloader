# DC 만화 다운로더



DCInside 게시글에서 고화질 이미지와 만화 번역본을 자동으로 수집, 추출 및 일괄 다운로드하도록 설계된 강력한 멀티스레드 도구입니다.



최신 CDN 방화벽 제한을 우회하도록 제작되었으며, 기본 첨부 파일, 본문 이미지 및 외부 호스트 링크(예: Imgur)를 완벽한 파일 무결성을 유지하며 처리합니다. 자동 크롤러, 스마트 다운로드 엔진, 백그라운드 로깅 데몬이 포함되어 있습니다.



---



##  상세 기능 목록



###  자동 게시판 크롤러 (`Run-Crawler.ps1`)

* **키워드 필터링:** 게시판 페이지에서 "번역" 키워드가 포함된 게시물을 스캔합니다.

* **설정 가능한 깊이 및 방향:** 스캔할 페이지 수(`MaxPages`)와 정렬 순서(과거순/최신순)를 설정할 수 있습니다.

* **스마트 리스트 관리:** 새로 발견된 URL을 `download_list.txt`의 `[automatic_urls]` 아래에 자동으로 추가하며, 기존 수동 항목 및 실패한 URL(`#RETRY`)을 보존합니다.

* **자동 핸드오프:** 크롤링 완료 후 즉시 다운로드 엔진을 실행합니다.



###  핵심 다운로드 엔진 (`Start-Downloader.ps1`)

* **지능형 403 Forbidden 우회:** 지역별 라우팅에 따른 이미지 차단을 우회합니다. 네이티브 파일(`data-fileno`)을 감지하면 숨겨진 첨부 파일 엔드포인트(`download.php`)를 통해 방화벽을 무력화합니다.

* **유니버설 링크 지원:** `data-tempno`를 통해 외부 호스트 이미지(Imgur 등)를 감지하고, 해당 서버의 차단을 피하기 위해 HTTP `Referer` 헤더를 동적으로 제거합니다.

* **엄격한 본문 슬라이싱:** `gallview_contents`부터 `reply_box`까지의 HTML 경계를 엄격히 적용합니다. 본문만 분리하여 하단의 썸네일 그리드나 추천 게시물이 다운로드되는 것을 방지합니다.

* **매직 바이트(Magic Byte) 검증:** 파일을 임시 `.tmp` 바이너리로 다운로드한 후, 원시 16진수 헤더를 읽어 `.jpg`, `.png`, `.gif`, `.webp` 확장자를 정확하게 할당합니다.

* **멀티스레드 실행:** PowerShell Job을 사용하여 여러 파일을 동시에 다운로드합니다. 최대 동시 다운로드 수는 설정에서 변경 가능합니다.

* **상태 재개 및 정리:**

  * **자동 정리:** 새 다운로드를 시작하기 전, 이전 작업에서 남은 좀비 `.tmp` 파일을 자동으로 삭제합니다.

  * **스마트 건너뛰기:** 기존 이미지 파일을 엄격히 확인하여 이미 완료된 콘텐츠는 중복 다운로드하지 않습니다.

  * **인터럽트 처리:** `Ctrl+C` 중단 시에도 백그라운드에서 완료된 작업의 통계를 안전하게 캡처하여 기록합니다.



###  백그라운드 로거 (`Background-Logger.ps1`)

* **Named Pipe IPC:** `\\.\pipe\DCMangaLogger`를 통해 통신하는 별도의 데몬으로 실행됩니다.

* **JSON 구조화 로그:** 메인 다운로드 스레드를 방해하지 않고 `activity_log.json`에 상세한 로그를 비동기식으로 기록합니다.

##  설정 및 요구 사항



### 요구 사항

* **운영 체제:** Windows 10 또는 Windows 11.

* **환경:** PowerShell 5.1 (기본) 또는 PowerShell 7 (Core).

* **네트워크:** 활성 인터넷 연결.



### 파일 구조

모든 파일이 동일한 디렉토리에 있는지 확인하세요:

```text

/

├── launch.bat                 # 메인 실행 파일 - 이것을 실행하세요!

├── config.yaml                # 환경 설정 파일

├── download_list.txt          # 자동/수동 처리 URL 저장소

├── Run-Crawler.ps1            # 게시판 스캔 스크립트

├── Start-Downloader.ps1       # 멀티스레드 다운로드 엔진

└── Background-Logger.ps1      # 로그 기록용 데몬

```



---



##  사용 방법



1. **스위트 실행:** `launch.bat` 파일을 더블 클릭합니다.

2. **메인 메뉴 옵션:**

   * **`1. Run Auto-Crawler & Downloader`**: 새 게시물을 스캔하여 리스트를 업데이트하고 즉시 다운로드를 시작합니다.

   * **`2. Run Manual Downloader`**: 크롤러를 건너뛰고 단일 URL을 붙여넣어 즉시 다운로드합니다.

   * **`3. Exit`**: 프로그램을 안전하게 종료하고 로거 프로세스를 해제합니다.

##  상세 설정 (`config.yaml`)



| 설정 항목 | 설명 |

| :--- | :--- |

| **BoardUrl** | 크롤링할 DCInside 게시판 리스트 URL. |

| **MaxPages** | 크롤러가 스캔할 페이지 수. |

| **CrawlOrder** | `0` = 과거 게시물부터, `1` = 최신 게시물부터. |

| **KeepUnfinishedLinks** | `False` 설정 시 성공한 링크를 리스트에서 자동 제거. |

| **DNSAutoRepair** | `True` 설정 시 네트워크 오류 발생 시 DNS 캐시를 초기화. |

| **UseProxy** | 시스템 프록시 우회 여부 (직접 연결 시 `False` 권장). |

| **RateLimitSeconds** | IP 차단 방지를 위한 게시물 간 대기 시간 (초). |

| **MaxConcurrentDownloads** | 동시 다운로드 작업 수 (권장: `10-15`). |

| **DownloadDir** | 이미지가 저장될 경로 (절대/상대 경로 지원). |

| **LogPath** | `activity_log.json` 로그 파일 경로. |

| **LogLevel** | `"Verbose"` (모든 로그) 또는 `"Error"` (실패만 기록). |

| **ShowProgressBar** | PowerShell 기본 다운로드 바 표시 여부. |

| **RenameFilesSequential** | `True` 설정 시 이미지를 `001.jpg` 등으로 번호 매김. |

| **ForceLegacyMode** | PowerShell v7이 있어도 v5.1 사용을 강제. |
